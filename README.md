## Endpoints

`GET /blog`

- mongodb에서 blog 관련 데이터를 읽어옵니다.

`POST /blog`

- 컬렉션 삭제후 blog 관련 데이터를 다시 입력합니다.

`PUT /blog`

- yml파일 내용이 바뀌면 컬렉션에 업데이트 하도록 합니다.

`GET /youtube`

- mongodb에서 youtube 관련 데이터를 읽어옵니다.

`POST /youtube`

- 컬렉션 삭제후 youtube 관련 데이터를 다시 입력합니다.
